Add scalebars to axes Adds a set of scale bars to ax, matching the size to the ticks of the plot and optionally hiding the x and y axes

Parameters:
  • ax – the axis to attach ticks to
  • matchx,matchy – if True, set size of scale bars to spacing between ticks if False, size should be set using sizex and sizey params
  • hidex,hidey – if True, hide x-axis and y-axis of parent
  • **kwargs – additional arguments passed to AnchoredScaleBars
Returns:

created scalebar object
