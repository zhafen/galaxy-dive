get_enclosed_mass(positions, snum, hubble_param, mt_halo_id=0)[source]¶

Get the mass enclosed at the positions given. Particles must be inside a halo, and the enclosed mass is only the mass of that halo.

Parameters:
  • positions (array-like, (n_positions,3)) – Positions in physical kpc you would like to find the mass interior to. Positions should be relative to the simulation box coordinates.
  • snum (int) – Snapshot at which to get the enclosed mass.
  • hubble_param (float) – The hubble parameter, H0/100, needed for converting to and from AHF’s units.
  • mt_halo_id (int) – Merger tree halo ID for the halo you want to use for finding the enclosed mass.
Returns:

The mass of Merger Traced halo mt_halo_id interior to the positions given. A value of np.nan indicates that the position is either outside the halo or too far in (and the results aren’t converged).
Return type:

array-like, (n_positions,)
