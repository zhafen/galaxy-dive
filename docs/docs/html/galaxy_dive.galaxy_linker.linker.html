  • type_of_halo_id (str) – If ‘halo_id’ then this is the halo_id at a given snapshot. If ‘mt_halo_id’ then this is the halo_id according to the merger tree.

  • supplementary_data (bool) – If True, also return supplementary data according to the keys in self.supplementary_data_keys

Shape ( n_particles, ).

The ID of the least massive substructure the particle’s part of. In the case of using the ‘mt_halo_id’, this is the ID of the most massive merger tree halo the particle’s part of. If it’s -2, then that particle is not part of any halo, within radial_cut_fraction*length_scale .
