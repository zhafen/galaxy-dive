galaxy_dive.plot_data.plotting.add_colorbar(fig_or_ax, color_object, method='fig', ax_location=[0.9, 0.1, 0.03, 0.8], **kw_args)[source]¶

Add a colorbar to a figure(fig) according to an object on the figure(color_object). Added to the right side of the figure. Or add a colorbar to the right of an axis.

Args:
fig_or_ax (figure or axis object): The object to add the colorbar to. color_object (object): The object providing the colorscheme, i.e. what you want the colorbar to provide a map for. method (str): If ‘fig’, add to the right of the figure. If ‘ax’, add to the right of the axis. ax_location (list): If adding to a figure, this is where and how large the axis will be. **kw_args will be passed to colorbar()
